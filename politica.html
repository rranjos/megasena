Política de Privacidade

Última atualização: 18 de novembro de 2025

Bem‑vindo ao Simulador MegaSena. Esta Política de Privacidade explica como coletamos, usamos, armazenamos e protegemos suas informações quando você usa nosso aplicativo, serviço hospedado em GitHub Pages ou arquivos estáticos relacionados.

1. Controlador de dados

O controlador dos dados pessoais coletados por este App é o desenvolvedor responsável pelo projeto. Substitua os dados de contato abaixo pelos seus:

  • Nome/Empresa: Dr. ByteCode
  • E‑mail para contato: dr.bytecode@gmail.com

2. Coleta de dados

O Simulador MegaSena não coleta nem armazena dados pessoais dos usuários. Todas as simulações são executadas localmente no seu dispositivo e nenhuma informação é enviada a servidores externos.

3. Finalidades do processamento

Finalidades do processamento

Usamos os dados apenas para os seguintes fins:

  • Prover e manter o funcionamento do App;
  • Melhorar a experiência do usuário e corrigir bugs;
  • Responder a solicitações, dúvidas e feedback enviados por você;
  • Gerar estatísticas de uso anônimas para entender tendências de uso (caso análises estejam ativas).

4. Base legal

Se você estiver no Brasil, a base legal aplicável inclui o seu consentimento para processamentos específicos (por ex., envio de e‑mail quando você se cadastrar para receber novidades) e o legítimo interesse para operações de diagnóstico e segurança. Para residentes na União Europeia, o processamento seguirá o Regulamento Geral de Proteção de Dados (GDPR) quando aplicável.

5. Compartilhamento de dados

Não vendemos, alugamos nem comercializamos seus dados pessoais. Podemos compartilhar informações com:

  • Prestadores de serviços contratados para operar o App (ex.: provedores de analytics, serviços de e‑mail) — apenas quando necessário e com contratos que exijam proteção adequada dos dados.
  • Autoridades públicas, se houver obrigação legal ou determinação judicial.

6. Armazenamento e segurança

Tomamos medidas técnicas e organizacionais razoáveis para proteger seus dados contra perda, acesso não autorizado, alteração ou divulgação. No entanto, nenhum método de transmissão ou armazenamento eletrônico é 100% seguro — recomendamos cuidado ao compartilhar informações sensíveis.

Se você deseja que excluamos quaisquer dados pessoais que você nos enviou por formulário ou e‑mail, contate o e‑mail listado na seção "Contato" abaixo.

7. Retenção dos dados

Reteremos dados pessoais apenas pelo tempo necessário para cumprir as finalidades descritas nesta política, a menos que períodos de retenção mais longos sejam exigidos por lei. Dados técnicos anônimos para análise podem ser mantidos por um período adicional para fins estatísticos.

8. Direitos dos titulares (LGPD / GDPR)

Dependendo da sua jurisdição, você pode ter direitos como:

  • Acesso aos seus dados;
  • Retificação de dados incorretos;
  • Eliminação (direito ao esquecimento);
  • Portabilidade dos dados;
  • Restrição de tratamento e objeção a certos tipos de processamento;
  • Retirada do consentimento quando o processamento se baseou nele.

Para exercer esses direitos, entre em contato pelo e‑mail informado.

9. Menores de idade

O App não é destinado a menores de 13 anos. Não coletamos intencionalmente informações pessoais de menores. Se você é pai/mãe ou responsável e acredita que seu filho nos forneceu dados pessoais, contate‑nos para solicitar a exclusão.

10. Uso de serviços de terceiros e GitHub Pages

O App pode utilizar serviços de terceiros (ex.: provedores de hospedagem como GitHub Pages, provedores de analytics, serviços de terceiros para envio de e‑mail). Esses provedores têm suas próprias políticas de privacidade e termos — recomendamos que revise as políticas desses serviços. Ao hospedar no GitHub Pages, os acessos podem ser registrados pelo GitHub conforme sua própria política.

11. Alterações nessa Política

Podemos atualizar esta Política de Privacidade ocasionalmente. Quando fizermos alterações significativas, publicaremos a nova versão nesta página e atualizaremos a data de "Última atualização". Recomendamos que revise esta página periodicamente.

12. Contato

Se tiver dúvidas sobre esta Política de Privacidade ou quiser exercer seus direitos, entre em contato:

  • E‑mail: seu-email@exemplo.com
